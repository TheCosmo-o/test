=======================================
        NASA TERMINAL SIMULATOR
=======================================

Opis:
------
Projekt to interaktywny symulator terminala
w stylu NASA. Użytkownik loguje się do systemu
przy użyciu specjalnych danych dostępowych,
a następnie widzi sekwencję startową misji
kosmicznej, komunikaty systemowe oraz
symulowane błędy i ostrzeżenia.

Funkcjonalności:
-----------------
- Ekran logowania z polami:
  * Astronaut ID
  * Launch Authorization Code
- Animowane wpisy w terminalu
  (symulacja logów startowych NASA).
- Sekwencja „wishes” = kolejne kroki misji:
  np. sprawdzanie systemów, odliczanie,
  start rakiety.
- Losowe komunikaty błędów/alertów
  po wpisywaniu komend w terminalu.
- Stylizowany prompt np.:
  commander@apollo11:~$

Instrukcja uruchomienia:
-------------------------
1. Pobierz wszystkie pliki projektu:
   - index.html
   - assets/css/style.css
   - assets/js/script_login.js
   - assets/js/script_terminal.js
   - assets/img/ (grafiki, tło NASA)

2. Otwórz plik index.html w przeglądarce.

3. Zaloguj się używając danych:
   - Astronaut ID: a.mendyk
   - Launch Authorization Code: INF.03
   (domyślne dane – można zmienić w pliku script_login.js)

4. Po poprawnym logowaniu pojawi się
   animowany terminal z sekwencją startową.
